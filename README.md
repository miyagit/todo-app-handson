# todo-app-handson

タスク管理アプリケーションのハンズオンプロジェクトです。

## 概要

このプロジェクトは、シンプルなTODO管理アプリケーションです。
ユーザーは日々のタスクを効率的に管理し、生産性を向上させることができます。

## 技術スタック

このアプリケーションは以下の技術で実装されています：

- **HTML** - アプリケーションの構造定義
- **CSS** - スタイリングとレイアウト
- **JavaScript** - ロジック処理とインタラクション
- **LocalStorage** - ローカルデータの永続化

## 機能

- タスクの追加・編集・削除
- タスクの完了状態管理
- LocalStorageを使用したデータ保存

## セットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/miyagit/todo-app-handson.git
```

2. プロジェクトディレクトリに移動
```bash
cd todo-app-handson
```

3. ブラウザでHTMLファイルを開く
```bash
open index.html
```

## 使用方法

1. タスク入力欄にタスク内容を入力
2. 追加ボタンをクリックしてタスクを追加
3. チェックボックスをクリックして完了状態を切り替え
4. 削除ボタンをクリックしてタスクを削除

## ライセンス

このプロジェクトはハンズオン学習用です。
